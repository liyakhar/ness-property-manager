## 📁 Data Storage:

Your data is stored in these files:
- **Properties**: `data/properties.json`
- **Tenants**: `data/tenants.json`
- **Updates**: `data/updates.json`
- **Images**: `data/images/`

## 🔗 API Endpoints:

- **Properties**: `/api/free-storage/properties`
